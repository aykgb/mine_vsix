# mine_vsix

## Steps to Create an Extension

1. Create a directory for the extension.
2. Add a `package.json` file in the directory.
3. Run `vsce package` and `vsce publish`.

## Login

Run the following script to log in:

```sh
./login.sh
```

The token can be found in the `.env` file.

## Publish

Run the following script to publish:

```sh
./publish.sh
```
